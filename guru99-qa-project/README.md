# Guru99 Banking QA Project

This project is based on the Guru99 Banking System SRS.

This folder contains my QA practice work during my own learning journey.

It demonstrates manual software testing by creating test scenarios and test cases based on real requirements.

---

## 📌 Project Overview
- Requirement analysis from SRS
- Test Scenario design
- Test Case creation
- Functional and negative testing
- Role-based testing (Manager & Customer)
